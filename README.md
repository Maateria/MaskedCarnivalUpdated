# MaskedCarnivale (Updated)

Updated build of [ProjectMimer/MaskedCarnivale](https://github.com/ProjectMimer/MaskedCarnivale) for current Dalamud.

## Install

Download `latest.zip` from the [Releases](https://github.com/Maateria/MaskedCarnivalUpdated/releases) page, extract it into your Dalamud `devPlugins` folder, then load it from the Dalamud plugin installer (Dev tab).
